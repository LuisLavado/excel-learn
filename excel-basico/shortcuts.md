
## Atajos de taclado para Excel:
- `Ctrl + A`/`Ctrl + E`/`Ctrl + *`/`Ctrl + Shift + *`: Selecciona toda la información en un solo movimiento.
- `Ctrl + Flechas`: Navegar entre celdas con datos
- `Ctrl + Shift + Flechas`: Seleccionar hasta el final del rango
- `Ctrl + 1` = Abre el grupo Formato de Celdas
- `Ctrl + Z`/`Ctrl + Y`: Deshacer y Rehacer
- `Doble Click`/`Alt + H + O + I`: Deshacer y Rehacer
- `Ctrl + +` / `Ctrl + Shift + +`: Insertar fila o columna
- `Ctrl + -`: Eliminar fila o columna
- `Ctrl + F`/`Ctrl + B`: Abre el menú de buscar
- `Ctrl + L`/`Ctrl + H` = Abre el menú de Buscar y Remplazar
- `Ctrl + Shift + $/%`: Formato moneda o porcentaje
- `Ctrl + F1`: Muestra o oculta barra de herramientas
- `Ctrl + J`: Replica información que está arriba de.
- `F4`: Repetir última acción
- `Ctrl + RePag/AvPag`/`Ctrl + Shift + RePag/AvPag`: Mover entre hojas
- `Ctrl + Enter`: Aplicar en múltiples celdas/Replica una información
